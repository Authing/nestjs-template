## Installation

```bash
$ yarn
```

## Commands

### Running the app

```bash
# run in dev environment
$ npm run start:dev

# run in test environment
$ npm run start:dev

# run in production environment
$ npm run start:prod
```

### Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
